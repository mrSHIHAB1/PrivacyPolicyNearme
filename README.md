# Near Me — Privacy Policy (React)

Minimal Vite + React app that displays the Near Me privacy policy for app-store submissions.

Getting started

1. Install dependencies

```bash
cd react-privacy
npm install
```

2. Run development server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
```

Files created

- `src/PrivacyPolicy.jsx` — privacy content
- `src/App.jsx` — app shell
- `index.html`, `package.json`, `src/main.jsx`, `src/index.css`
